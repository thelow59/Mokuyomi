Features

  • Mokuro OCR overlay — real DOM text, tappable for Yomitan
  • Cross-device sync — progress and immersion time via server
  • Immersion timer — tracks time spent reading per volume
  • Touch navigation — pinch-to-zoom, swipe-to-pan
  • Library management — series cards, per-volume progress
  • Zero dependencies — pure Python stdlib, no pip install

Quick start

1. Put your Mokuro-processed manga in the manga/ folder.

2. Run Mokuyomi (Start Menu shortcut or double-click the launcher).

3. Open http://localhost:8090 in your browser.

Or run from source: git clone → python3 mokuyomi.py
